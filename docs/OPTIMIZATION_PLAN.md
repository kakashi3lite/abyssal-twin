## Branch A — Performance

### A1. N+1 Correlated Subquery *(SELECTED — implemented)*
**File:** `cloudflare/src/routes/fleet.ts:32`
**Severity:** High
For every vehicle row in the fleet status query, D1 executes a separate correlated
subquery to find the latest state vector ID. With N vehicles this is N+1 total DB
operations per request.

```sql
-- Before (N+1):
LEFT JOIN state_vectors sv ON sv.vehicle_id = v.id
  AND sv.id = (SELECT MAX(id) FROM state_vectors WHERE vehicle_id = v.id)

-- After (1 derived-table scan):
LEFT JOIN (
  SELECT vehicle_id, MAX(id) AS max_id
  FROM state_vectors
  GROUP BY vehicle_id
) latest ON latest.vehicle_id = v.id
LEFT JOIN state_vectors sv ON sv.id = latest.max_id
```

**Estimated improvement:** O(N) → O(1) additional queries per fleet status poll.

### A2. Unconditional Anomaly Polling
**File:** `cloudflare/pages/src/components/AnomalyPanel.tsx:41-46`
**Severity:** High
`setInterval(loadAnomalies, 30_000)` fires unconditionally every 30s regardless of
tab visibility or whether anomaly data has changed. Estimated 2,880 requests/day/user
on the satellite-constrained link.

**Recommended fix (not yet implemented):**
Add `document.visibilityState` guard and ETag/`If-None-Match` caching headers.

`AUVMesh` recreates `boxGeometry` / `capsuleGeometry` on every parent render because
the component is not wrapped in `React.memo`. Wrap component and move static geometry

## Branch B — Cleanliness

### B1. `window as any` Global Namespace Pollution *(SELECTED — implemented)*
**File:** `mission-control/src/main.ts:57-68`
**Severity:** Medium-High
Nine class methods assigned to `window` via `as any` casts to support inline
`onclick="..."` HTML attributes. Breaks TypeScript coverage, pollutes global scope,
and couples markup to runtime globals.

**Fix applied:** Replaced with direct `addEventListener` bindings for static buttons
and event delegation (`data-action` attributes) for dynamically rendered widget
controls. Zero `any` casts remain.

---

### B2. `any` Types in Demo Engine
**File:** `mission-control/src/demo-data.ts:41, 309, 335, 342`
**Severity:** Medium
`Map<number, any>` and `(data: any) => void` listener types. Define a concrete
`VehicleState` interface and type-safe event map.

---

### B3. God Class — DashboardManager (784 lines)
**File:** `mission-control/src/main.ts`
**Severity:** Medium
Handles widgets, connections, charts, export, settings, and theme in one class.
Candidate for extraction: `WidgetManager`, `ConnectionManager`, `ChartManager`.

---

### B4. Duplicated Vehicle Profiles
**Files:** `cloudflare/src/simulation-engine.ts:71-74` vs `mission-control/src/demo-data.ts:30-34`
**Severity:** Low
Identical vehicle fleet definitions (names, IDs, base positions, depths) in two
places. Extract to a shared config JSON or `packages/fleet-config`.

---

### B5. `createAUVCard` innerHTML Injection Pattern
**File:** `mission-control/src/main.ts:474`
**Severity:** Low
`content.innerHTML = fleet.vehicles.map(v => createAUVCard(v)).join('')` —
data originates from controlled API, not user input, so XSS risk is low. However
the pattern is fragile; prefer `createElement` + `appendChild` for safety.

## Branch C — Security

### C1. CORS Wildcard on Simulation SSE Endpoint *(SELECTED — implemented)*
**File:** `cloudflare/src/index.ts:161`
**Severity:** Critical
`/api/v1/simulate` set a manual `"Access-Control-Allow-Origin": "*"` header,
bypassing the global Hono `cors()` middleware. Any origin could read the live
telemetry SSE stream.

**Fix applied:** Removed the manual header. CORS is now controlled exclusively by
the global middleware at line 28, which can be restricted per-environment via Hono's
`cors({ origin: ... })` option.

---

### C2. Incomplete JWT Signature Verification
**File:** `cloudflare/src/middleware/auth.ts:95-113`
**Severity:** High
`validateAccessJWT` decodes the JWT payload and checks expiry but does not verify
the cryptographic signature against Cloudflare Access public keys. Commented code
shows the intended implementation path (`crypto.subtle.verify` against
`<issuer>/cdn-cgi/access/certs`).

**Recommended fix (separate security PR):**
```typescript
const certsUrl = `${payload.iss}/cdn-cgi/access/certs`;
const certsResp = await fetch(certsUrl);
const { keys } = await certsResp.json<{ keys: JsonWebKey[] }>();
// import key + verify signature with crypto.subtle
```

---

### C3. Global Hono `cors()` Defaults to Wildcard
**File:** `cloudflare/src/index.ts:28`
**Severity:** Medium
`app.use("*", cors())` with no options defaults to `Access-Control-Allow-Origin: *`.
Configure with `cors({ origin: env.ALLOWED_ORIGIN })` and add `ALLOWED_ORIGIN` to
`wrangler.toml` per environment.
